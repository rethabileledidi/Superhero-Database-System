# 🦸 One Kick Heroes Academy – Superhero Database System

A C# Windows Forms application developed for the PRG2782 Project Module. The system was designed to manage superhero trainee records for the One Kick Heroes Academy by storing, updating, and generating reports for hero data.

The project uses file handling, Windows Forms, and GitHub version control to simulate a real-world database management system.

---

# 📌 Features

- Add new superhero records
- Automatically calculate hero rank and threat level
- View all superheroes in a DataGridView
- Update superhero information
- Delete superhero records
- Generate summary reports
- Save data using text files
- Git & GitHub version control integration

---

# 🧰 Technologies Used

- C#

- .NET Framework
- Text File Handling
- Git
- GitHub
- Visual Studio

---

# 🖥 System Functionality

The application allows users to:

- Store superhero details such as:
  - Hero ID
  - Name
  - Age
  - Superpower
  - Exam Score

- Automatically determine:
  - Hero Rank (S, A, B, C)
  - Threat Level

- Generate reports showing:
  - Total superheroes
  - Average age
  - Average exam score
  - Heroes per rank
